The left part of equation (10.3.2) shows the important idea needed to find a unit speed parameterization: we will need to pick a transformation \(\alpha(t)\) to satisfy
\begin{equation*} \vecmag{\frac{d\vr}{du}} \left\vert\frac{d\alpha}{dt}\right\vert =1\text{.} \end{equation*}
Algebraically, this is very difficult to write out in closed form. However, it will be possible because the arc length traveled as a function of the parameter cannot decrease. If we make a two-dimensional plot with the parameter value on the horizontal axis and the arc length traveled on the vertical axis, then we have a graph that passes the horizontal line test.
