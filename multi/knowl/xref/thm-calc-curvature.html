Theorem 10.5.6.

If \(\vr\) is a parameterization of a curve \(C\text{,}\) and if \(\vr'(t)\neq \vzero\) and \(\vr''(t)\) exists, then the curvature \(\kappa\) of \(C\) can be calculated by
\begin{equation*} \kappa(t) = \frac{\left\Vert \vT'(t) \right\Vert}{ \left\Vert \vr'(t) \right\Vert} \end{equation*}
ðŸ”—
Further, we can efficiently calculate curvature in terms of the velocity and acceleration functions by
\begin{equation} \kappa = \frac{\Vert \vr'(t) \times \vr''(t) \Vert}{\Vert \vr'(t) \Vert^3}= \frac{\Vert \vv(t) \times \va(t) \Vert}{\Vert \vv(t) \Vert^3}\tag{10.5.1} \end{equation}

ðŸ”—

Proof.

We will give the outline of a proof of the first result here in order to emphasize that this result comes from a chain rule argument related to the change of parameters going to a unit speed parameterization.
ðŸ”—
Let \(\vr(t)\) be some parameterization of our curve \(C\text{.}\) Remember that in order to calculate the curvature, \(\vT\) will need to have an input corresponding to the parameter of the unit speed parameterization, which means the input of the \(\vT\) will need to measure the arclength of \(C\) to the point of interest. We will use a function \(s(t)\) with input being the parameter of \(\vr(t)\) and the output of \(s(t)\) will be the arclength on \(C\) to the point \(\vr(t)\text{.}\) Note that \(s(t)\) is computed using TheoremÂ 10.3.5 and that the derivative \(\frac{ds}{dt}\) will be \(\Vert\vr'(t)\Vert\text{.}\)
ðŸ”—
We can write \(\vT\) as a function of the parameter \(t\) by the composition \(\vT(t)=\vT(s(t))\text{.}\) Taking the derivative of this composition with respect to \(t\) will utilize a chain rule argument.
\begin{equation*} \frac{d\vT}{dt}=\frac{d}{dt}\left(\vT(s(t))\right)=\frac{d\vT}{ds} \frac{ds}{dt} \end{equation*}
Note here that \(\frac{d\vT}{dt}=\vT'\) is the same derivative of \(\vT\) we have seen a few times.
ðŸ”—
Because \(\frac{ds}{dt}\) is a scalar valued function we can divide both sides of the equation by it to get
\begin{equation*} \frac{\vT'}{\frac{ds}{dt}}=\frac{d\vT}{ds} \end{equation*}
Using our idea above from TheoremÂ 10.3.5, we can substitute \(\frac{ds}{dt}= \Vert\vr'(t)\Vert\) and take the magnitude of each side to get the following
\begin{equation} \left\Vert \frac{\vT'}{\Vert\vr'(t)\Vert} \right\Vert = \left\Vert\frac{d\vT}{ds} \right\Vert =\kappa\tag{10.5.2} \end{equation}
ðŸ”—
If this chain rule argument seems a little technical to you, there is a nice conceptual interpretation of the equation above. Specifically, if we want to convert a measurement like curvature or the derivative of \(\vT\) between the unit speed parameterization and any other parameterization, we literally need to adjust by the speed. Equation (10.5.2) shows that the adjustment between the derivative of \(\vT\) with respect to the unit speed parameter and another parameter is to divide by the variable speed.
