Theorem 11.7.6. Calculating Directional Derivatives.

Given a differentiable function \(f = f(x,y)\) and a unit vector \(\vu = \langle u_1, u_2 \rangle\text{,}\) the directional derivative \(D_{\vu}f(x,y)\) can be computed by
\begin{equation} D_{\vu}f(x,y) = f_x(x,y) u_1 + f_y(x,y) u_2\text{.}\tag{11.7.2} \end{equation}
