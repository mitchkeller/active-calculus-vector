Example 9.8.12.

In this example, we will look at how to convert the points with rectangular coordinates \(P=(-2,3)\) and \(Q=(-3,-2)\) into polar coordinates. Note here that \(P\) is in the second quadrant and \(Q\) is in the third quadrant.

Using the conversion equations for \(P=(-2,3)\text{,}\) we see that
\begin{equation*} r^2 = (-2)^2 + (3)^2 \ \ \ \ \ \tan(\theta) = -\frac{3}{2} \end{equation*}
which means we need to pick \(r\) to be \(\sqrt{13}\) or \(-\sqrt{13}\text{.}\) We will need to pick an appropriate angle for these possible \(r\)-values that satisfy \(\tan(\theta) = -\frac{3}{2}\text{.}\) There are angles in the fourth and second quadrants that will satisfy \(\tan(\theta) = -\frac{3}{2}\text{,}\) namely \(\arctan(-\frac{3}{2})\) and \(\pi +\arctan(-\frac{3}{2})\text{.}\) You could also choose any angle coterminal to either of these angles. This is not a situation where we can arbitrarily combine one of the choices of \(r\) will work with any choice of \(\theta\text{.}\) We must interpret these choices together to get a description for the correct location.
ðŸ”—
If we choose \(r=\sqrt{13}\text{,}\) then we need to select the angle that corresponds to the second quadrant, \(\pi + \arctan(-\frac{3}{2})\text{,}\) to get polar coordinates \(P=(r,\theta)=(\sqrt{13},\pi+\arctan(-\frac{3}{2}))\text{.}\) If we want to not have to use the complementary angle to \(\arctan(-\frac{3}{2})\text{,}\) then we need to use the negative value of \(r\) to get \(P=(r,\theta)=(-\sqrt{13},\arctan(-\frac{3}{2}))\text{.}\) FigureÂ 9.8.13 illustrates these two different polar coordinates that identify the point \(P\)

Now we will look at how to convert \(Q=(-3,-2)\) into polar coordinates. Similar to our arguments above, we find that \(Q=(-3,-2)\) implies that \(r^2=9+4=13\) and \(\tan(\theta)=\frac{-2}{-3}\text{.}\) If we want to consider a positive value for \(r\) and a value of \(\theta\) between \(0\) and \(2\pi\text{,}\) then we get \(r=\sqrt{13}\) and \(\theta=\arctan(\frac{2}{3})+\pi \text{.}\) We could also consider a negative value for \(r\) and a value of \(\theta\) between \(0\) and \(2\pi\text{,}\) which would yield \(r=-\sqrt{13}\) and \(\theta=\arctan(\frac{2}{3}) \text{.}\) FigureÂ 9.8.14 illustrates these two perspectives on finding polar coordinates for \(Q\)

Remember that inverse trig functions have limited domains and ranges, the \(r\)-coordinate can be positive or negative, and polar coordinates do not have unique values for a given point.
