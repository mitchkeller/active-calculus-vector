In single variable calculus, we learned that a differentiable function is locally linear. In other words, if we zoom in on the graph of a differentiable function around a point, the graph will look like the tangent line to the function at that point. In FigureÂ 11.5.1 you can look at the graph of a curve and zoom in around a point on that curve to see that eventually the graph of the function will look linear. You can use the sliders above the plot to change how closely zoomed in the plot will be or change the location on the curve that you are examining. You should visually verify at a several points that this function is differentiable because at any point shown, the graph is locally linear.
