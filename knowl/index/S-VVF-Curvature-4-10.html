Definition 10.5.5.

If \(C\) is a smooth space curve and \(s\) is the parameter for the unit speed parameterization of \(C\text{,}\) then the curvature, \(\kappa\text{,}\) of \(C\) is
\begin{equation*} \kappa = \kappa(s) = \vecmag{ \frac{d \vT}{ds} } \end{equation*}
Remember that the parameter for the unit speed parameterization \(C\) is the arclength traveled along the curve. Note that curvature is denoted with \(\kappa\text{,}\) the Greek lowercase letter “kappa”.
