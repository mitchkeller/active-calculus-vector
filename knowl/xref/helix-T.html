Example 10.4.4.

For our first example, we will look computing the unit tangent vector of a helix. In particular, we will look at the helix traced out by \(\vr(t)=\langle \cos(t),\sin(t),\frac{t}{\pi}\rangle \text{.}\)

Based on our parameterization, we have velocity vector given by \(\vv(t)=\langle -\sin(t),\cos(t),\frac{1}{\pi} \rangle \text{,}\) which has speed given by
\begin{equation*} \text{speed}(t) = \Vert\vv(t)\Vert = \sqrt{(-\sin(t))^2+(\cos(t))^2+(\frac{1}{\pi})^2}= \sqrt{1+\frac{1}{\pi^2}} \end{equation*}
One of the useful and unique ideas for this example is that we have a constant speed. No matter what parameter value we look at, the speed is always \(\sqrt{1+\frac{1}{\pi^2}} \text{.}\) Using DefinitionÂ 10.4.3, we can see that
\begin{equation*} \vT(t)= \frac{\vv(t)}{\Vert{\vv(t)}\Vert}= \frac{1}{\sqrt{1+\frac{1}{\pi^2}}}\langle -\sin(t),\cos(t),\frac{1}{\pi} \rangle \end{equation*}
If \(t=\frac{\pi}{2}\text{,}\) then \(\vT(\frac{\pi}{2})= \frac{1}{\sqrt{1+\frac{1}{\pi^2}}}\langle -1,0,\frac{1}{\pi} \rangle\text{.}\) If \(t=\frac{5\pi}{4}\text{,}\) then \(\vT(\frac{5\pi}{4})= \frac{1}{\sqrt{1+\frac{1}{\pi^2}}}\langle \frac{\sqrt{2}}{2},-\frac{\sqrt{2}}{2},\frac{1}{\pi} \rangle\text{.}\) In FigureÂ 10.4.6, \(\vT(\frac{\pi}{2})\) is plotted in red and \(\vT(\frac{5\pi}{4})\) is plotted in blue. Both of these vectors are unit length and tangent to the curve in the direction of travel for that instant.

Note that we can also find \(\vT(\frac{\pi}{2})\) by taking \(\vv(\frac{\pi}{2}) = \langle -1,0,\frac{1}{\pi} \rangle \) and shrinking \(\vv\) to be a unit vector. Because our definitions and theorems work for all \(t\)-values, we can apply them at a specific \(t\)-value, which has the added advantage that computations tend to be manageable when done by hand. The downside to this calculation is that you only have information for the one parameter value This will only useful for finding information at a fixed \(t\)-value because you are not concerned with how the measurement is done for other parameter values.
