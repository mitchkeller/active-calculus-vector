Example 10.3.6.

In this example, we look a few simple paths and how to write a parameterization that moves with unit speed. For our first path, we will consider the linear path given by starting at the point \((x_0,y_0,z_0)\) when \(t=a\) and going to the point \((x_1,y_1,z_1)\) when \(t=b\text{.}\) This line will be described by the parameterization \(\vr(t) = \langle x_0+\frac{t-a}{b-a}(x_1-x_0),y_0+\frac{t-a}{b-a}(y_1-y_0),z_0+\frac{t-a}{b-a}(z_1-z_0) \rangle \text{.}\) This parameterization will have speed
\begin{equation*} \Vert \vr'(t)\Vert = \sqrt{(\frac{x_1-x_0}{b-a})^2+(\frac{y_1-y_0}{b-a})^2+(\frac{z_1-z_0}{b-a})^2} \end{equation*}
Note that the speed of this parameterization is constant, so for simplicity we will call this constant speed \(v_{line}\text{.}\)
In order to change this parameterization to move with speed 1, we will need to travel \(v_{line}\) times slower, which can be accomplished by replacing \(t\) in our parameterization with \(\frac{t}{v_{line}}\text{.}\) In other words, we adjusted the parameter used to make sure the speed with respect to new parameter will be one. This adjustment was easy to do because the amount we needed to change the parameter did not change at different times.
Algebraically, we can characterize this change by thinking of \(\vr(u)\) as the “old” parameterization with parameter \(u\) and we are looking to replace \(u\) with some function \(\alpha(t)\) so that \(\Vert \vr'(t)\Vert=1\) for all \(t\text{.}\) In our linear path case, our choice of \(\alpha(t) = \frac{t}{v_{line}}\) meant that applying the chain rule to \(\Vert \vr'(t)\Vert\) gives
\begin{equation*} \Vert \vr(\alpha(t)) \Vert= \left\Vert \frac{d\vr}{du}\right\Vert \left\vert\frac{d\alpha}{dt}\right\vert=v_{line} \frac{1}{v_{line}} =1 \end{equation*}
