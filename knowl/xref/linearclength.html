Example 10.3.6.

In this example, we look at a linear path and explore how to write a parameterization that moves with unit speed. We will look at the parameterization of the line through \((1,2,3)\) and \((-2,3,1)\text{.}\) We are using these values to make the calculations less abstract but there is nothing special about these values and the methods will work for any linear parameterization.
An easy to find parameterization will have the base point \((1,2,3)\) and direction vector given by the difference between our points, \(\langle -2-1,3-2,1-3\rangle = \langle -3,1,-2 \rangle\text{.}\) So we start with the parameterization of the line given by \(\vr(t)=\langle 1,2,3\rangle + t \langle -3,1,-2 \rangle\) and we convert this to a parameterization with unit speed. The speed of our parameterization will be
\begin{equation*} \vecmag{\vr(t)}=\vecmag{\frac{d\vr}{dt}}=\vecmag{\langle -3,1,2\rangle}=\sqrt{14} \end{equation*}
So we need to adjust our parameterization to travel \(\sqrt{14}\) times slower, which can be accomplished by replacing \(t\) in our parameterization with \(\frac{t}{\sqrt{14}}\text{.}\) In other words, we adjusted the parameter used to make sure the speed with respect to the new parameter will be one. This adjustment was easy to do because the amount we needed to change the parameter did not change at different times.
Our proposed unit speed parameterization is \(\vr_1(t)= \langle 1,2,3\rangle + \frac{t}{\sqrt{14}} \langle -3,1,-2 \rangle\) which will have speed
\begin{equation*} \vecmag{\vr_1(t)}=\vecmag{\frac{1}{\sqrt{14}}\langle -3,1,2\rangle}=\frac{\sqrt{14}}{\sqrt{14}}=1 \end{equation*}
Algebraically, we can characterize this change by thinking of \(\vr(u)\) as the “old” parameterization with parameter \(u\) and we are looking to replace \(u\) with some function \(\alpha(t)\) so that \(\Vert \vr'(t)\Vert=1\) for all \(t\text{.}\) In our linear path case, our choice of \(\alpha(t) = \frac{t}{\sqrt{14}}\) meant that applying the chain rule to \(\Vert \vr'(t)\Vert\) gives
\begin{equation} \Vert \vr(\alpha(t)) \Vert= \left\Vert \frac{d\vr}{du}\right\Vert \left\vert\frac{d\alpha}{dt}\right\vert=\sqrt{14} \frac{1}{\sqrt{14}} =1\tag{10.3.2} \end{equation}
