\(R\) into subrectangles with the lines \(x=0\text{,}\) \(x=0.5\text{,}\) \(x=1\text{,}\) \(x=1.5\text{,}\) and \(x=2\) and the lines \(y=0\text{,}\) \(y=1\text{,}\) \(y=2\text{,}\) \(y=3\text{,}\) and \(y=4\text{.}\) Then we have 16 subrectangles, each of which we denote \(R_{(a,b)}\text{,}\) where \((a,b)\) is the location of the lower-left corner of the subrectangle. We want to find a lower bound and an upper bound for the volume above each subrectangle. The lower bound for the volume of \(R_{(a,b)}\) is
\begin{equation*} 0.5 ( \mbox{Min of } f \mbox{ on } R_{(a,b)}) \end{equation*}
because the area of \(R_{(a,b)}\) is \(0.5\cdot 1=0.5\text{.}\) The function \(f(x,y) = 1+2xy\) increases with both \(x\) and \(y\) over the whole region \(R\text{.}\) Thus,
\begin{equation*} \mbox{Min of } f \mbox{ on } R_{(a,b)} = f(a,b) = 1+2ab, \end{equation*}
because the minimum on each subrectangle is at the corner closest to the origin. Similarly,
\begin{equation*} \mbox{Max of } f \mbox{ on } R_{(a,b)} = f(a+0.5,b+1) = 1 + 2(a+0.5)(b+1). \end{equation*}
So we have
\begin{equation*} \mbox{Lower sum } = \sum_{(a,b)} 0.5(1+2ab) = 0.5 \sum_{(a,b)} 1+2ab = 8 + 0.5\sum_{(a,b)}2 ab \end{equation*}
Since \(a=0, 0.5, 1, 1.5\) and \(b= 0, 1, 2, 3\text{,}\) expanding this sum gives
\begin{equation*} \mbox{Lower} = 8 + 0.5 (0 + 0 + 0 + 0 + 0 + 1 + 2 + 3 + 0 + 2 + 4 \end{equation*}
\begin{equation*} + 6 + 0 + 3 + 6 + 9) = 26. \end{equation*}
Similarly, we can compute the upper sum:
\begin{equation*} \mbox{Upper sum }= \sum_{(a,b)}0.5(1+2(a+0.5)(b+1)) = 0.5\sum_{(a,b)}(1+2(a+0.5)(b+1)) \end{equation*}
\begin{equation*} = 8 + 0.5\sum_{(a,b)}(a+0.5)(b+1) = 58. \end{equation*}
