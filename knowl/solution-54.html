Solution 11.1.5.3.1.

(a) If we take the partition of \(R\) consisting of just \(R\) itself, we get
\begin{equation*} \mbox{Lower bound }= \mbox{min}_R f\cdot A_R=0\cdot (2-0)(2-0)=0. \end{equation*}
Similarly, we get
\begin{equation*} \mbox{Upper bound } = \hbox{max}_R f\cdot A_R = 2\sqrt{3}\cdot (2-0)(2-0)=8\sqrt{3}. \end{equation*}
(b) The estimates asked for are just the upper and lower sums. We partition \(R\) into subrectangles \(R_{(a,b)}\) of width 1 and height 1, where \((a,b)\) is the lower-left corner of \(R_{(a,b)}\text{.}\) The subrectangles are then \(R_{(0,0)}\text{,}\) \(R_{(1,0)}\text{,}\) \(R_{(0,2)}\text{,}\) and \(R_{(2,2)}\text{.}\) Then we find the lower sum
\begin{equation*} \mbox{Lower } =\sum_{(a,b)}A_{R_{(a,b)}}\cdot\min_{R_{(a,b)}}f = \sum_{(a,b)} 1\cdot (\mbox{Min of }f \mbox{ on }R_{(a,b)}). \end{equation*}
The minimum occurs at the lower-left corner of each rectangle, so
\begin{equation*} \mbox{Lower } = 1\cdot( f(0,0) + f(1,0) + f(0,1) + f(1,1) ) = 4\sqrt{3}. \end{equation*}
Similarly, the upper sum is
\begin{equation*} \mbox{Upper } = 1\sum_{(a,b)}(\mbox{Max of }f \mbox{ on }R_{(a,b)}), \end{equation*}
so
\begin{equation*} \mbox{Upper } = 1(f(2,2)+f(4,2)+f(2,4)+f(4,4)) = 1(3\sqrt3 + 2\sqrt{6}). \end{equation*}
The upper sum is an overestimate and the lower sum is an underestimate, so we can get a better estimate by averaging them to get \(1(2\sqrt3 + \sqrt6)\text{.}\)
