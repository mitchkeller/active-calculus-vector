(a) The probability is
\begin{equation*} \int_0^1\int_{1/2}^1 {2 \over 3} (x + 2y) \,dx\,dy = \int_0^1 {2\over3} ({1\over2}x^2 + 2xy)\bigr|^1_{1/2}\,dy \end{equation*}
\begin{equation*} = \int_0^1 {2\over3} \left[({1\over2} + 2y) - ({1\over 8} + {}y)\right]\,dy \end{equation*}
\begin{equation*} = {2\over3}\int_0^1 (\frac{3}{8} + y) \, dy = {2\over3} \left(\frac{3}{8} y + \frac{1}{2} y^2\right)\bigr|_0^1 \end{equation*}
\begin{equation*} = {2\over3} \left(\frac{7}{8}\right) = \frac{7}{12}. \end{equation*}
(b) It is easier to calculate the probability that \(x \lt \frac1{2} + y\) does not happen, that is, the probability that \(x \ge \frac1{2} + y\text{,}\) and subtract it from 1. The probability that \(x \ge \frac1{2} + y\) is
\begin{equation*} \int_{0}^{1/2}\int_{y+1/2}^{1} {2\over3} (x+2y) \,dx\,dy = \int_{0}^{1/2} (\frac13\,x^2 + \frac43\,xy) \bigr|_{y+(1/2)}^{1} \,dy \end{equation*}
\begin{equation*} = \int_{0}^{1/2} \frac13\left(1 - (\frac1{2} + y)^2\right) + \frac43(y)(1 - (\frac1{2} + y)) \,dy \end{equation*}
\begin{equation*} = \int_{0}^{1/2} -\frac53\,y^2 + \frac{1}{3} y + \frac{1}{4} \, dy \end{equation*}
\begin{equation*} = (-\frac59\,y^3 + \frac{1}{6}\, y^2 + \frac{1}{4}\,y)\bigr|_{0}^{1/2} \end{equation*}
\begin{equation*} = \frac{7}{72}. \end{equation*}
Thus, the probability that \(x \lt \frac1{2}+y\) is \(1-(\frac{7}{72}) = \frac{65}{72}\text{.}\)
