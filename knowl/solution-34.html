Solution 10.4.5.5.1.

(a) Since \(z\) is a linear function of \(x\) and \(y\) with slope -4 in the \(x\) direction, and slope 1 in the \(y\) direction, we have:
\begin{equation*} z = -4 x + 1 y + c. \end{equation*}
We can write an equation for changes in \(z\) in terms of changes in \(x\) and \(y\text{:}\)
\begin{equation*} \Delta z = (-4 (x + \Delta x) + 1(y + \Delta y) + c) - (-4 x + 1 y + c) = -4 \Delta x + 1 \Delta y. \end{equation*}
Since \(\Delta x = -0.4\) and \(\Delta y = 0.2\text{,}\) we have
\begin{equation*} \Delta z = -4 (-0.4) + 1(0.2) = 1.8. \end{equation*}
So a \(-0.4\) change in \(x\) and a \(0.2\) change in \(y\) produces a \(1.8\) change in \(z\text{.}\) (b) As we know that \(z=7\) when \(x=3\) and \(y=5\text{,}\) the value of \(z\) when \(x=2.9\) and \(y=5.1\) will be
\begin{equation*} z = 7 + \Delta z = 7 - 4\Delta x + 1\Delta y. \end{equation*}
where \(\Delta z\) is the change in \(z\) when \(x\) changes from 3 to 2.9 and \(y\) changes from 5 to 5.1. We have \(\Delta x = 2.9 - 3 = -0.1\) and \(\Delta y = 5.1 - 5 = 0.1\text{.}\) Therefore, when \(x=2.9\) and \(y=5.1\text{,}\) we have
\begin{equation*} z = 7 - 4 \cdot(-0.1) + 1\cdot(0.1) = 7.5. \end{equation*}
