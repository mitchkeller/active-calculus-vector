Solution 12.7.6.6.1.

(a) Since \(\mbox{curl } \vec F = 8x\,\mathit{\vec i}-8z\,\mathit{\vec k}\text{,}\) we know by the curl test that \(\vec F\) is not a gradient field. (b) Since \(\mbox{curl } \vec G = -2z\,\mathit{\vec i}+\left(y-5z\right)\,\mathit{\vec j}-\left(2x+z\right)\,\mathit{\vec k}\) we know by the curl test that \(\vec G\) is not a gradient field. (c) Since \(\mbox{curl } \vec H = \boldsymbol{0}\)and \(\vec H\) is defined everywhere we know by the curl test that \(\vec H\) is a gradient field.
