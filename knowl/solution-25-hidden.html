(b) Similarly, if \((x,y)\) tends to \((0,0)\) along the curve \(y = 5 x\text{,}\) then
\begin{equation*} f(x,y) = f(x,5 x) = \frac{4x^{2}-\left(5x\right)^{2}}{x^{2}+\left(5x\right)^{2}} = \frac{-21}{26}, \end{equation*}
and so
\begin{equation*} \lim_{x\to0} f(x,5 x) = \frac{-21}{26}. \end{equation*}
Because we have two different values for these limits, we know that the function doesnâ€™t have a limit at \((0,0)\text{.}\) No matter how close we are to the origin, there will be points on \(f(x,y)\) that are close to \(\frac{-12}{17}\) and points that are close to \(\frac{-21}{26}\text{,}\) whereas for the limit to exist the function must get close to a single value. (c) We can see this more generally by considering the limit as \((x,y)\) tends to \((0,0)\) along the curve \(y = m x\text{.}\) Then
\begin{equation*} f(x,y) = f(x,m x) = \frac{4-m^{2}}{1+m^{2}}, \end{equation*}
so that
\begin{equation*} \lim_{x\to0} f(x,m x) = \frac{4-m^{2}}{1+m^{2}}, \end{equation*}
and we see for every different value of \(m\) we will get a different limiting value for the function. Thus the function does not have a limit at \((0,0)\text{.}\)
